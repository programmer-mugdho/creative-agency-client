<strong>Creative Agency</strong> is a website for an agency where a customer can <strong>order</strong> for a service which is saved in a <strong>database</strong>. An <strong>admin</strong> can see orders in the <strong>dashboard</strong> and <strong>mark</strong> them if itâ€™s <strong>done</strong> or <strong>pending</strong>. A customer can write a <strong>review</strong> which will be visible in the <strong>review section</strong>. An admin can <strong>add</strong> a new <strong>service</strong> from the <strong>dashboard</strong> which will be visible in the <strong>service section</strong> so that a customer can ask for that service.
